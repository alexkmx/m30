# Simulador App Streaming de Música

Copia App de streaming de musica en la que se consume una API para realizar busqueda de discografia por artista. Se puede agregar y eliminar canciones en lista de favoritos.

# Cuadro de busqueda
Se tiene un cuadro para ingresar el nombre del artista realiza la busqueda de discografía(si es que el Artista esta en la BD).

# Encontrando discografía
Al mostrar la discografía, se puede dar clic en cualquier albúm para que muestre las canciones que contiene

# Agregar o eliminar canciones de la lista
Se pueden agregar y eliminar canciones a una lista
